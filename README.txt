Mon projet porte sur une application de chat ,de discussion instantannée entre
 un utilisateur et plusieurs autres.
Joint à ce fichier ,tous les fichiers des differentes classes utilisées dans un dossier clientserveur2 qu'il 
faudra juste lancer sur Eclipse.
->Vous devriez aussi ajouter la librairie Apache : org.apache.commons.lang3.StringUtils

NB: tout le code à été developpé sur Eclipse

De ce faite mon code est constitué de 4 classes diffrentes :

1 . Class Client qui sera notre classe chargée d'envoyer les sockets au 
serveur pour pouvoir communiquer.

2.Class Serveur ,elle representera un serveur recevant des requetes de
connection de la part d'un ou de plusieurs Clients et comportant la liste des 
communications.(serveur-client)

3.Class Message qui sera une classe nous permettant d'instantier un message 
qui transite entre un client et un serveur.

4.Class serveurs qui est une classe implementant la communication entre un serveur et un client grace aux 
clientSocket.
